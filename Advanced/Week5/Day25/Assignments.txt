1.  How to add CSS class to html tag using jQuery?
Ans: addClass() - Adds one or more classes to the selected elements
     removeClass() - Removes one or more classes from the selected elements
     toggleClass() - Toggles between adding/removing classes from the selected elements
     
2.  How to apply effect based on width property?
Ans: The animate() method performs a custom animation of a set of CSS properties. 
     Syntax: (selector).animate({styles},speed,easing,callback)
