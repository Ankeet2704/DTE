1.  How to add events to html elements at runtime  using javascript?
Ans: The addEventListener() method attaches an event handler to the specified element.
     The addEventListener() method attaches an event handler to an element without overwriting existing event handlers.
     You can add many event handlers to one element.

2.  How to use keypress, keyup, keydown events?
Ans: There are three different keyboard events in JavaScript:
     keydown: Keydown happens when the key is pressed down, and auto repeats if the key is pressed down for long.
     keypress: This event is fired when an alphabetic, numeric, or punctuation key is pressed down.(Some browsers no longer support keypress event)
     keyup: Keyup happens when the key is released.
